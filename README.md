# Mood-to-Playlist 🎵

Type a mood or situation — get a Spotify-style playlist generated by AI.

## Examples
- `"3am can't sleep"` → lo-fi chill playlist
- `"driving through rain"` → moody indie rock set
- `"Monday morning hype"` → energetic pop bangers

## Stack

| Layer | Tech |
|-------|------|
| Frontend | React + TypeScript (Vite) |
| Backend | FastAPI (Python) |
| AI | OpenAI Chat API |
| Cache | Redis |

## Project Structure

```
mood-to-playlist/
├── backend/          # FastAPI Python backend
│   ├── main.py
│   └── requirements.txt
├── frontend/         # React + TypeScript frontend (Vite)
│   ├── src/
│   └── ...
└── README.md
```

## Getting Started

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

Create `backend/.env`:

```env
OPENAI_API_KEY=your_openai_api_key
REDIS_URL=redis://localhost:6379
```
