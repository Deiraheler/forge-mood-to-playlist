# Mood-to-Playlist 🎵

Type a mood or situation — get a Spotify-style playlist generated by AI.

## Examples
- `"3am can't sleep"` → lo-fi chill playlist
- `"driving through rain"` → moody indie rock set
- `"Monday morning hype"` → energetic pop bangers

## Stack

| Layer | Tech |
|-------|------|
| Frontend | React + TypeScript (Vite) |
| Backend | FastAPI (Python) |
| AI | OpenAI Chat API |
| Cache | Redis |

## Project Structure

```
mood-to-playlist/
├── backend/          # FastAPI Python backend
│   ├── main.py
│   └── requirements.txt
├── frontend/         # React + TypeScript frontend (Vite)
│   ├── src/
│   └── ...
└── README.md
```

## Getting Started

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Verifying the Setup

Once both servers are running:

1. **Frontend** — open http://localhost:5173 → see the default React starter page
2. **Backend health** — open http://localhost:8000/api/health → `{"status":"ok"}`
3. **Proxy** — from the frontend dev server: `curl http://localhost:5173/api/health` → `{"status":"ok"}` (Vite proxies `/api/*` → `http://localhost:8000`)

## Environment Variables

Create `backend/.env`:

```env
OPENAI_API_KEY=your_openai_api_key
REDIS_URL=redis://localhost:6379
```
